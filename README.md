# 🚀 Mini VS Code - Web Code Editor

A modern, browser-based **Mini VS Code** built using **React**. This project replicates the core functionality of a lightweight code editor with a clean UI and powerful features like live preview, file management, and multi-language support.

---

## ✨ Features

### 🧑‍💻 Code Editor

* Multi-tab editing system
* Syntax highlighting (HTML, CSS, JavaScript, JSON)
* Auto-closing brackets & tags
* Line numbers and formatting support
* Search & Replace functionality

### 📁 File Management

* File explorer sidebar
* Create, rename, and delete files
* Drag & drop file upload
* Download project as ZIP

### ⚡ Live Development

* Real-time preview (HTML/CSS/JS)
* Built-in console for output & errors
* Basic linting support

### 🎨 UI/UX

* VS Code-inspired layout
* Clean modern design (no glassmorphism/neumorphism)
* Dark/Light theme toggle
* Smooth animations and responsive design

### 🧠 Smart Features

* Auto-save using localStorage
* Keyboard shortcuts (Ctrl+S, Ctrl+F, etc.)
* Word & character counter
* File icons based on extensions

---

## 🛠️ Tech Stack

* **React (Vite)**
* **Monaco Editor**
* **Tailwind CSS / CSS**
* **JavaScript (ES6+)**
* **localStorage API**

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone <your-repo-url>
```

2. Navigate to project folder:

```bash
cd mini-vscode
```

3. Install dependencies:

```bash
npm install
```

4. Run the project:

```bash
npm run dev
```

5. Open in browser:

```
http://localhost:5173
```
